## 🔧 curl 명령어 문제

curl 테스트에서 JSON 파싱 오류가 발생하는 이유:
- Windows cmd에서 JSON 이스케이프 문제
- 한글 인코딩 문제

**해결책**: 브라우저에서 테스트하면 문제 없음!
